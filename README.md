# DeFi Token Swap Basic

This project helps beginners learn how decentralized exchanges work and how tokens can be swapped using blockchain.

You will learn:
- What DeFi is
- How token swapping works
- How to connect a wallet
- How to call a smart contract
- How to read transaction status

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/defi-token-swap-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- swap.js = Token swap functions
- web3.js = Blockchain connection
- deploy.js = Contract deploy script
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Select tokens
- Swap tokens
- Show transaction hash
- Simple DeFi flow

## Learning Goal
This project explains how decentralized exchanges work in a simple way.
